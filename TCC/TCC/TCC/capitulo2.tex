Atualmente, existem diversos tipos de processadores de textos com os mais diversos recursos de forma a tornar a editoração de textos o mais fácil possivel, mesmo para textos com layouts mais complexos. Porém, cada processador de texto possui suas características que os fazem ser categorizados. Basicamente existem duas categorias de programas para processamento de texto \cite{ApostilaUSP}. 

\begin{itemize}
 \item \textbf{Estilo visual}
 \item \textbf{Estilo lógico}
\end{itemize}

\section{ESTILO VISUAL}

Neste estilo, existe um menu na tela apresentando os recursos que eles oferecem, sendo possível selecioná-los pelo mouse. Os softwares que se encaixam nesta categoria oferecem a interface conhecida como \textit{What you is what you get - WYSIWYG} (o que você vê é o que você obtém). Isso significa que a forma em que está formatado o documento durante a sua edição, será a maneira que ele estará ao ser impresso. Existem diversos processadores de texto de estilo visual, sejam eles softwares proprietários ou softwares livres, dentre os quais podemos citar:

\begin{itemize}
\item \textbf{Softwares Livres}: OpenOffice Writer, LibreOffice Writer, AbiWord;
\item \textbf{Softwares Proprietários}: Word, Bloco de Notas e WordPad.
\end{itemize}


\section{ESTILO LÓGICO}
Nos editores que adotam o estio lógico, o processamento é feito em duas fases distintas:
\begin{itemize}
\item Os comandos de formatação e o texto a ser impresso são escritos em um arquivo fonte com o uso de um editor. Isso não impede que o editor utilizado tenha menus, de forma a facilitar acesso a alguns comandos e ferramentas.
\item Após digitado, o documento é compilado gerando um arquivo de saida com o texto já formatado. 
\end{itemize}

Um exemplo de processador de texto baseado no estilo lógico é o Latex, o qual será abordado no capítulo a seguir.
